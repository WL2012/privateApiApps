# privateApiApps

知识点:

1. 获取iPhone中安装的APP列表
2. 获取每个APP的相关信息
3. 获取APP图标
4. 打开APP

文档介绍请见: [私有API的使用](http://markmiao.com/2016/12/28/%E7%A7%81%E6%9C%89API%E7%9A%84%E4%BD%BF%E7%94%A8/)
